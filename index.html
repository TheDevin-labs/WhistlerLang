WhistlerLang
Reference

A dynamically typed, scientific scripting language designed for clarity and expressiveness. Built for data, math, and fast iteration.

Dynamically typed Scientific built-ins v26.0 stable

𝑥 Variables

Declare variables with the let keyword. WhistlerLang is dynamically typed — no type annotation required.

let x    = 10          -- integer
let pi   = 3.14159     -- float
let name = "Whistler"  -- string
let flag = true        -- boolean
Dynamic typing — the type of a variable is inferred at runtime. You can rebind a name to a different type, but this is generally discouraged for clarity.

▶ Output

Use the say keyword to print any value to the console. No parentheses needed — say is a keyword, not a function.

say "Hello World"   -- Hello World
say x              -- prints value of x → 10
say 2 + 2          -- evaluates then prints → 4
say flag           -- true

[ ] Arrays

Arrays hold ordered sequences of values, declared with square brackets. Indexing starts at 0.

let nums  = [1, 2, 3, 4, 5]
let words = ["hi", "world"]

let first = nums[0]   -- 1  (first element)
let third = nums[2]   -- 3  (third element)
Zero-indexed — the first element is always at index 0, the last at len(arr) - 1.

⊞ Matrices

Matrices are two-dimensional arrays. Access elements with two indexes: mat[row][col].

let mat = [[1, 2, 3],
           [4, 5, 6],
           [7, 8, 9]]     -- 3×3 matrix

let center = mat[1][1]  -- row 1, col 1 → 5
let corner = mat[0][2]  -- row 0, col 2 → 3

fn Functions

Declare functions with fn, a name, parameters in (), and ->. The body goes in { }. The last expression is automatically returned.

fn add(a, b) -> {
    a + b          -- implicitly returned
}

fn square(x) -> {
    x * x
}

fn greet(name) -> {
    say "Hello"
    say name
}

-- Calling functions
let result = add(3, 4)  -- result = 7
say square(5)          -- 25
greet("World")          -- Hello \n World
Implicit return — no return keyword needed. The final expression in the function body is the return value.

? Conditionals

Use if, elif, and else for branching logic. elif is checked when the preceding if was false.

let score = 85

if score >= 90 {
    say "Grade: A"
} elif score >= 75 {
    say "Grade: B"   -- this branch runs
} elif score >= 60 {
    say "Grade: C"
} else {
    say "Grade: F"
}

↻ Loops

Two loop forms: range(n) to iterate n times (0 through n−1), or iterate directly over an array.

-- range loop: i goes 0, 1, 2, 3, 4
for i in range(5) {
    say i
}

-- array loop
let data = [10, 20, 30]
for item in data {
    say item        -- 10, 20, 30
}

-- Comments

WhistlerLang supports single-line comments only. A comment begins with -- and extends to the end of the line.

-- This is a full-line comment
let x = 10  -- This is an inline comment
No block comments — multi-line comments are not supported. Prefix each line with --.

✦ Full program

A complete scientific program combining variables, functions, statistics, and conditionals.

  science_demo.wl 
say "=== WhistlerLang Science Demo ==="

-- Store temperature measurements
let measurements = [98.6, 97.9, 99.1, 98.3, 100.2]

say "Measurements:"
say measurements

-- Calculate statistics
let avg_temp  = mean(measurements)
let deviation = std(measurements)

say "Average temperature (F):"
say avg_temp

say "Standard deviation:"
say deviation

-- Unit conversion function
fn celsius(f) -> {
    (f - 32.0) * 5.0 / 9.0
}

say "Average in Celsius:"
say celsius(avg_temp)

-- Status check
if avg_temp > 99.5 {
    say "Status: Fever detected"
} elif avg_temp > 98.9 {
    say "Status: Slightly elevated"
} else {
    say "Status: Normal"
}
